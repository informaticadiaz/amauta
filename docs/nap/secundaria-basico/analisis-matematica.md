## Propósitos Generales del Ciclo Básico

La escuela ofrecerá situaciones de enseñanza que promuevan:

1. Confianza en las propias posibilidades para resolver problemas y formularse interrogantes
2. Concepción de la Matemática como consecuencia necesaria de la aplicación de relaciones
3. Disposición para defender puntos de vista, considerar ideas de otros, debatir y elaborar conclusiones
4. Interpretación de información (textos, tablas, fórmulas, gráficos, expresiones algebraicas)
5. Elaboración de procedimientos para resolver problemas
6. Interpretación y producción de textos con información matemática
7. Comparación y análisis de validez de producciones
8. Producción e interpretación de conjeturas y afirmaciones generales
9. Explicitación de conocimientos con distintas representaciones
10. Comprensión y uso de la organización decimal del sistema de numeración
11. Reconocimiento y uso de números racionales y sus representaciones
12. Uso y explicitación de operaciones en distintos campos numéricos
13. Uso de jerarquías y propiedades de las operaciones
14. Análisis y uso de procedimientos de cálculo (mental, escrito, exacto, aproximado)
15. Producción y validación de enunciados sobre relaciones numéricas
16. Reconocimiento y análisis de variaciones funcionales
17. Reconocimiento y uso de expresiones algebraicas
18. Uso y explicitación de propiedades de figuras y cuerpos geométricos
19. Producción y análisis de construcciones geométricas
20. Producción y validación de conjeturas geométricas
21. Uso y explicitación de sistemas de unidades de medida
22. Análisis y uso de procedimientos para estimar y calcular medidas
23. Interpretación y uso de nociones básicas de estadística
24. Reconocimiento y uso de nociones de probabilidad

## Ejes Temáticos Organizadores

Los NAP se organizan en 4 ejes:

1. **En relación con el Número y las Operaciones**
2. **En relación con el Álgebra y las Funciones**
3. **En relación con la Geometría y la Medida**
4. **En relación con la Probabilidad y la Estadística**

## NAP Primero/Segundo Año

### Eje 1: Número y Operaciones

| Contenido                     | Descripción                                                                                      |
| ----------------------------- | ------------------------------------------------------------------------------------------------ |
| **Números enteros (Z)**       | Interpretar, registrar, comunicar y comparar en diferentes contextos                             |
|                               | - Como número relativo (temperaturas, nivel del mar)                                             |
|                               | - A partir de la resta de dos naturales (juegos de cartas, pérdidas y ganancias)                 |
|                               | Comparar números enteros y hallar distancias en la recta numérica                                |
| **Números racionales (Q)**    | Interpretar el número racional como cociente                                                     |
|                               | Usar diferentes representaciones (fraccionarias, decimales, notación científica, recta numérica) |
|                               | Argumentar sobre equivalencia y elegir representación adecuada                                   |
| **Propiedades**               | Analizar diferencias y similitudes entre Z y Q (orden, discretitud, densidad)                    |
| **Operaciones en Z**          | Interpretar modelos para suma, resta, multiplicación, división y potenciación                    |
| **Potenciación y radicación** | Usar potenciación (exponente entero) y radicación en Q; analizar propiedades                     |
| **Operaciones en Z**          | Analizar como extensión de las elaboradas en N                                                   |
| **Estrategias de cálculo**    | Seleccionar tipo de cálculo (mental/escrito, exacto/aproximado, con/sin calculadora)             |
|                               | Evaluar razonabilidad del resultado                                                              |
| **Jerarquía de operaciones**  | Usar jerarquía y propiedades en producción e interpretación de cálculos                          |
| **Divisibilidad**             | Explorar y enunciar propiedades en N (suma de múltiplos, transitividad)                          |

### Eje 2: Álgebra y Funciones

| Contenido                      | Descripción                                                                  |
| ------------------------------ | ---------------------------------------------------------------------------- |
| **Relaciones entre variables** | Interpretar en tablas, gráficos y fórmulas (regularidades, proporcionalidad) |
| **Variaciones uniformes**      | Modelizar y expresar eligiendo representación adecuada                       |
| **Proporcionalidad directa**   | Explicitar y analizar propiedades (variación uniforme, origen en cero)       |
| **Fórmulas geométricas**       | Producir y comparar fórmulas para perímetros, áreas y volúmenes              |
| **Regularidades numéricas**    | Producir fórmulas en N y analizar equivalencias                              |
| **Afirmaciones algebraicas**   | Producir y analizar afirmaciones sobre propiedades de operaciones            |
| **Expresiones algebraicas**    | Transformar para obtener expresiones equivalentes                            |
| **Ecuaciones lineales**        | Resolver ecuaciones del tipo ax + b = cx + d                                 |
|                                | Usar como expresión de condición sobre conjunto de números                   |
|                                | Analizar conjunto solución (única, infinitas, sin solución)                  |

### Eje 3: Geometría y Medida

| Contenido                  | Descripción                                                                                                    |
| -------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **Lugares geométricos**    | Determinar puntos que cumplan condiciones de distancia                                                         |
|                            | Construir circunferencias, círculos, mediatrices y bisectrices                                                 |
| **Triángulos**             | Explorar construcciones y argumentar sobre condiciones de congruencia                                          |
| **Polígonos**              | Construir con regla no graduada y compás; justificar procedimientos                                            |
| **Ángulos**                | Formular conjeturas sobre relaciones (opuestos por el vértice, adyacentes, paralelas cortadas por transversal) |
|                            | Producir argumentos para validar a partir de propiedades del paralelogramo                                     |
| **Propiedades de figuras** | Analizar afirmaciones y argumentar sobre validez                                                               |
|                            | Reconocer límites de las pruebas empíricas                                                                     |
| **Teorema de Pitágoras**   | Analizar relaciones entre lados de triángulos (ternas pitagóricas)                                             |
|                            | Interpretar demostraciones basadas en equivalencia de áreas                                                    |
| **Medición**               | Estimar y calcular cantidades eligiendo unidad y forma conveniente                                             |
|                            | Reconocer inexactitud de toda medición                                                                         |
|                            | Usar notación científica para cantidades muy grandes o pequeñas                                                |
| **Cuerpos geométricos**    | Explorar relaciones entre área lateral y volumen                                                               |

### Eje 4: Probabilidad y Estadística

| Contenido                   | Descripción                                        |
| --------------------------- | -------------------------------------------------- |
| **Estadística descriptiva** | Organizar conjuntos de datos discretos y acotados  |
|                             | Analizar proceso de relevamiento de datos          |
| **Variables**               | Identificar variables cualitativas y cuantitativas |
|                             | Organizar datos y construir gráficos adecuados     |
| **Parámetros centrales**    | Interpretar significado de la media y el modo      |
| **Probabilidad**            | Comparar probabilidades de diferentes sucesos      |
|                             | Incluir casos con conteo ordenado (sin fórmulas)   |

| Contenido                    | Descripción                                                                   |

### Eje 2: Álgebra y Funciones

| Contenido                      | Descripción                                                                          |

| **Transformaciones isométricas** | Explorar variaciones en triángulos y cuadriláteros                             |

1. **Numéricas**: expresiones fraccionarias, decimales, notación científica

## Enfoque Didáctico

El documento enfatiza:

1. **Resolución de problemas**: Como eje central del aprendizaje matemático
2. **Argumentación**: Avanzar de pruebas empíricas a argumentos más generales
3. **Representaciones múltiples**: Usar y vincular diferentes formas de representación
4. **Modelización**: Elegir variables y determinar conjuntos de valores
5. **Validación**: Reconocer límites de las pruebas empíricas
6. **Cálculo reflexivo**: Seleccionar tipo de cálculo según situación

## Observaciones para Implementación

1. **Estructura clara por ejes**: 4 ejes bien definidos facilitan la organización curricular
2. **Progresión explícita**: Contenidos de 2°/3° amplían y profundizan los de 1°/2°
3. **Énfasis en argumentación**: Se busca superar el nivel empírico
4. **Integración de tecnología**: Se menciona explícitamente el uso de recursos tecnológicos
5. **Conexión entre representaciones**: Fundamental el paso entre formas de representación
6. **Estadística y probabilidad**: Incluidos como eje propio (no secundario)
7. **Geometría constructiva**: Construcciones con regla y compás como herramienta de aprendizaje
8. **Modelización**: Concepto clave para álgebra y funciones
9. **Notas al pie explicativas**: El documento incluye aclaraciones importantes
