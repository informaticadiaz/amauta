## Nota sobre la Organización Curricular

**IMPORTANTE**: A diferencia de otros NAP organizados por año, este documento está organizado por **DISCIPLINA**. Cada jurisdicción determina si el espacio curricular se desarrolla en 1, 2 o 3 años dentro del Ciclo Orientado.

Los NAP corresponden a:

- **3°, 4° y 5° Años** en Jurisdicciones con Nivel Secundario de **5 años**
- **4°, 5° y 6° Años** en Jurisdicciones con Nivel Secundario de **6 años**

## NAP Física

### Organización Temática

**Nota**: A diferencia de Biología y Química, Física no presenta ejes explícitos sino bloques temáticos interrelacionados.

| Bloque Temático    | Descripción                                    |
| ------------------ | ---------------------------------------------- |
| **Energía**        | Concepto unificador de la física               |
| **Modelos**        | Partícula, onda, campo                         |
| **Física Moderna** | Introducción a mecánica cuántica y relatividad |

### Contenidos de Energía

| Contenido                    | Descripción                                                      |
| ---------------------------- | ---------------------------------------------------------------- |
| Conservación de la energía   | Principio de conservación en sistemas mecánicos y termodinámicos |
| Transformaciones energéticas | Conversiones entre formas de energía                             |
| Degradación de la energía    | Segunda ley de la termodinámica, entropía                        |
| Energía y sociedad           | Fuentes de energía, impacto ambiental, sustentabilidad           |

### Contenidos de Modelos

| Contenido                   | Descripción                                                          |
| --------------------------- | -------------------------------------------------------------------- |
| Modelo de partícula         | Mecánica newtoniana, conservación del momentum                       |
| Modelo de onda              | Ondas mecánicas y electromagnéticas, interferencia, difracción       |
| Modelo de campo             | Campo eléctrico, campo magnético, campo gravitatorio                 |
| Interacciones fundamentales | Gravitatoria, electromagnética, nuclear fuerte y débil               |
| Electromagnetismo           | Relación entre electricidad y magnetismo, inducción electromagnética |
| Espectro electromagnético   | Naturaleza de la luz, ondas de radio, rayos X, etc.                  |

### Contenidos de Física Moderna

| Contenido                        | Descripción                                         |
| -------------------------------- | --------------------------------------------------- |
| Mecánica cuántica (introducción) | Dualidad onda-partícula, cuantización de la energía |
| Efecto fotoeléctrico             | Evidencia de la naturaleza corpuscular de la luz    |
| Modelo atómico cuántico          | Niveles de energía, orbitales                       |
| Relatividad especial (nociones)  | Dilatación temporal, contracción de longitudes      |

## Diferencias Estructurales con Otros NAP

| Aspecto                   | Otros NAP (ej: Matemática)    | Ciencias Naturales                          |
| ------------------------- | ----------------------------- | ------------------------------------------- |
| **Organización**          | Por año (3°/4°, 4°/5°, 5°/6°) | Por disciplina (Biología, Física, Química)  |
| **Distribución temporal** | Fija por año                  | Flexible (1, 2 o 3 años según jurisdicción) |
| **Ejes**                  | Comunes a todos los años      | Específicos de cada disciplina              |
| **Progresión**            | Explícita año a año           | Implícita dentro de cada disciplina         |

## Articulación con Ciclo Básico

| Disciplina   | Ciclo Básico                                    | Ciclo Orientado                                   |
| ------------ | ----------------------------------------------- | ------------------------------------------------- |
| **Biología** | Célula, sistemas del cuerpo humano, ecosistemas | Genética molecular, evolución, biotecnología      |
| **Física**   | Mecánica básica, energía, electricidad simple   | Campos, ondas, física moderna                     |
| **Química**  | Estructura de la materia, cambios químicos      | Estructura atómica avanzada, equilibrio, orgánica |

## Observaciones para Implementación

1. **Organización por disciplina**: El modelo de datos debe contemplar que estos NAP no tienen progresión por año sino por disciplina
2. **Flexibilidad jurisdiccional**: Las provincias deciden la distribución temporal (1, 2 o 3 años)
3. **Física sin ejes formales**: A diferencia de Biología y Química, Física se organiza por bloques temáticos
4. **Física moderna**: Incluye contenidos introductorios de mecánica cuántica y relatividad
5. **Biotecnología**: Contenido relevante y actual en Biología
6. **Química ambiental**: Conexión con problemáticas ambientales actuales
7. **Enfoque integrador**: Las tres disciplinas comparten el concepto de energía como tema transversal
8. **Modelización**: Énfasis en el uso de modelos científicos para explicar fenómenos
9. **Contextualización**: Relación ciencia-tecnología-sociedad-ambiente (CTSA)
10. **Articulación**: Clara continuidad con contenidos del Ciclo Básico
